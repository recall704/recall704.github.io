ingress-nginx-controller
2023-12-17

ingress-nginx-controller 一般来说一个环境就只安装一个，默认独占 80 和 443 端口。

理论上只需要部署在一个机器上就可以了，需要注意的是这里使用的是 Deployment ，我们应该指定运行在某个或某几个机器上。

  1. 升级规则应该修改为 Recreate
  2. 需要固定在一台或几台机器上(使用 VIP)
  3. 线上需要结合 DNS 服务器使用
  4. 本地调试可以结合 Chrome 插件 Host Switch Plus 来访问。
