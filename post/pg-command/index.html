postgresql常用命令

2023-12-25
  1. 连接数据库：psql -U 用户名 -d 数据库名 -h 主机名 -p 端口号
  2. 查看数据库：\l 或 SELECT datname FROM pg_database;
  3. 切换数据库：\c 数据库名
  4. 查看表：\dt 或 SELECT table_name FROM information_schema.tables WHERE table_schema='public';
  5. 查看表结构：\d 表名 或 SELECT column_name, data_type FROM information_schema.columns WHERE table_name='表名';
  6. 创建表：CREATE TABLE 表名 (列1 数据类型, 列2 数据类型, ...);
  7. 修改表：ALTER TABLE 表名 ...;
  8. 删除表：DROP TABLE 表名;
  9. 插入数据：INSERT INTO 表名 (列1, 列2, ...) VALUES (值1, 值2, ...);
